# Essenza — Vending Premium

Sitio web oficial de Essenza, servicio de vending premium en Guadalajara.

## Estructura

```
essenza-web/
├── index.html          ← Página principal
├── hersheys/
│   └── index.html      ← Propuesta Hershey's
├── assets/
│   ├── essenza_logo.svg
│   ├── essenza_icon.svg
│   └── essenza_logo_white.svg
└── README.md
```

## URLs

- **Principal**: `essenza.mx` (o el dominio configurado)
- **Propuesta Hershey's**: `essenza.mx/hersheys`

## Deploy

### Netlify
1. Conectar repo de GitHub
2. Build command: (ninguno)
3. Publish directory: `/`

### GitHub Pages
1. Settings → Pages
2. Source: Deploy from branch `main`
3. Folder: `/ (root)`
